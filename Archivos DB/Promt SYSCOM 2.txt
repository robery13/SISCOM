Proyecto: Sistema de Control de Medicación (SISCOM)

Descripción general del proyecto: 
El sistema SISCOM tiene como objetivo principal ayudar a los pacientes adultos y geriátricos en el control de su medicación diaria mediante alarmas programadas y registro de dosis. A través de una aplicación o sistema web, el paciente o su cuidador podrá ingresar los medicamentos con su respectiva dosis, frecuencia y horario. El sistema enviará recordatorios automáticos (alarmas) y permitirá marcar cuando una dosis ha sido tomada. Además, se generará un registro de cumplimiento que podrá ser visualizado por el cuidador o familiar para garantizar que el tratamiento se está siguiendo correctamente.

Herramientas y tecnologías utilizadas:
 - Lenguaje de programación: JavaScript (Node.js) 
- Base de datos: MySQL 
- Interfaz: HTML, CSS, JavaScript 
- Framework Backend: Express.js v5.1.0 
- Framework Frontend: Bootstrap 5.3.3 
- Entorno de desarrollo: Visual Studio Code 
- Control de versiones: GitHub 
- Notificaciones: Web Notifications API
 - Herramientas de gestión ágil: Trello
